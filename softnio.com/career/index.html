Job Description

  • Developing template using HTML, CSS and JavaScript.
  • Developing user-facing applications using React.js and/or Vue.js.
  • Updating existing templates with new pages and sections.
  • Responsible for the entire front-end development lifecycle.
  • Must ensure pixel-perfect, retina-ready and browser compatibility.
